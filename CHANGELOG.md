# Changelog

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/).

## [1.0.0] — 2026-07-03

### Добавлено

- Модульная структура: `config_manager.py`, `sync_core.py`, `ui_main.py`, `ui_settings.py`, `logger_config.py`; точка входа — `main.py`.
- Многопоточность: синхронизация выполняется в фоновом потоке, GUI остаётся отзывчивым.
- Чекбоксы активности пар в окне настроек; префиксы `[ACTIVE]` / `[INACTIVE]` в `config.cfg`.
- Подробное логирование: каждый скопированный файл (путь, размер, дата изменения).
- Скрипт `build_exe.cmd` для сборки автономного exe через PyInstaller.
- Готовые exe-сборки в GitHub Releases.

### Изменено

- **Логирование**: заменён `loguru` на стандартный `logging` (`logger_config.py`). Убрана внешняя зависимость — проект теперь использует только стандартную библиотеку Python.
- **Критерий копирования:** файл копируется только если в источнике он **новее по дате изменения** (`mtime`).
- **Скрипты запуска**: используют `venv\Scripts\python.exe` напрямую вместо `activate.bat`.
- **Сборка**: добавлена поддержка PyInstaller для создания автономного exe.

### Исправлено

- Ложное сообщение «Не заданы пути» для пар вида `Z:\R ===>>> R:\` (корень диска).
- Venv пересоздан с текущей версией Python.
- Зависание GUI при копировании больших файлов (вынесено в отдельный поток).
